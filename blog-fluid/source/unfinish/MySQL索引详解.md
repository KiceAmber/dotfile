# ☘️ 索引概述

索引是一种用于快速查找和检索数据的数据结构，本质是可以看作为一种排好序的数据结构

索引的作用相当于一本书的目录，通过索引可以快速查询数据库中的数据，

常见的索引结构有：`B树`、`B+树`、`Hash` 和 `红黑树`，而 MySQL 默认存储引擎 InnoDB 采用的则是 B+树

# 索引的优缺点

索引的优点：

- 使用索引可以大大提高查询数据的效率，这也是创建索引的主要原因
- 通过创建唯一索引，可以保证数据库表中每一行数据的唯一性
